🔧 HTML 使用方法

在 index.html 的 <head> 部分添加以下代码：

<!-- 现代浏览器 SVG favicon --> <link rel="icon" type="image/svg+xml" href="/favicon.svg"> <!-- 传统浏览器 fallback --> <link rel="icon" type="image/x-icon" href="/favicon.ico"> <!-- 不同尺寸的 favicon --> <link rel="icon" type="image/svg+xml" sizes="16x16" href="/favicon-16x16.svg"> <link rel="icon" type="image/svg+xml" sizes="32x32" href="/favicon-32x32.svg"> <link rel="icon" type="image/svg+xml" sizes="64x64" href="/favicon-64x64.svg"> <!-- PWA 应用图标 --> <link rel="apple-touch-icon" href="/logo192.svg"> <link rel="manifest" href="/manifest.json">
📝 注意事项
  • 所有图标都使用 SVG 格式，确保在任何分辨率下都清晰显示
  • 图标采用蓝色主题，与 Ticketradar 品牌色彩一致
  • SVG 格式支持现代浏览器，文件体积小，加载快
  • 如需 PNG 格式，可以使用在线工具将 SVG 转换为 PNG
🎨 图标设计说明

图标设计基于飞机主题，包含以下元素：

  • 机头：使用 #1a73e8 蓝色，代表主要品牌色
  • 机身：同样使用主品牌色，保持一致性
  • 机翼：使用 #1565c0 深蓝色，增加层次感
  • 尾翼：使用 #0d47a1 更深的蓝色，突出细节
  • 高光效果：使用 #4285f4 亮蓝色，增加立体感
